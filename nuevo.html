Cómo subir un repositorio paso a paso

  1. git init

    Crea un nuevo repositorio de Git en tu directorio actual.

  2. git add .

    Prepara los archivos modificados o nuevos moviéndolos al área de preparación (staging area). Esto le indica a Git qué cambios formarán parte de la próxima confirmación.

  3. git commit -m "Mensaje"

    Guarda en tu historial los archivos preparados previamente. La opción -m indica que escribirás un mensaje entre comillas para describir brevemente los cambios.

  4. git branch -M main

    Cambia el nombre de la rama principal por defecto a "main". Las ramas son bifurcaciones del código que te permiten experimentar sin romper el proyecto principal.

  5. git remote add origin https://github.com/...

    Vincula tu repositorio local a un servidor remoto (como GitHub o GitLab). La palabra "origin" es el alias predeterminado para esa URL.

  6. git push -u origin main

    Envía los cambios de tu rama local a la rama remota en GitHub y la configura como predeterminada.
